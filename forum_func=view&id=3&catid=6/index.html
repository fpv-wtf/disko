The pixelformat that is used in dfb_surface_init() depends on the pixelformat of your graphics- or videolayer that you are using. You can determine the native pixelformat by using dfblayer.
Internally we are using ARGB but if your layer only supports RGB16, the last blit from DirectFB can only be done by using this pixelformat.
