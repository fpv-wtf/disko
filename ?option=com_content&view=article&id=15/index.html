Doorbell  

Doorbell is a VOIP-based outdoor doorbell that can be run on SIP-based telephone systems (PBX). For more product information take a look at references.

The binary package was built on top of the board support package provided by emlix for the Arrow® BF527, which is based on the µClinux distribution for blackfin.

To compile the sources you can follow the installation instructions.

Package Description Size
Doorbell v1.02 binary package This is the last stable version of the doorbell application, precompiled for the EPC-BF527 platform. 4.3M
