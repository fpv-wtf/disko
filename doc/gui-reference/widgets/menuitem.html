MenuItem

Description

This element adds an item to a <menu>.

XML-Tag

<menuitem>

Attributes

Attribute Domain Default Value Description
widget.<name>.<attribute> element/attribute specific element/attribute specific If you use this construct you can set attributes of widgets which are located in the menu item template.

Remarks

This element can only be used within <mmsdialog> as a child of <menu>.

Examples

  • Create and fill a simple listbox
