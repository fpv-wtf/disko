VBox

Description

<hbox> and <vbox> elements are used to layout your dialogs.

The <vbox> specifies a box in which the child widgets are vertically arranged.

A vbox is a widget which cannot be selected and focused.

XML-Tag

<vbox>

Attributes

The vbox can only have name and size attributes, which are described here.

Remarks

This element can be used as child within windows and widgets.
It can have only one child widget.
