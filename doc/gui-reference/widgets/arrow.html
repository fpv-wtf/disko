Arrow

Description

Specifies an arrow. An arrow is a widget which can be selected but not focused.

XML-Tag

<arrow>

Attributes

Attribute Domain Default Value Description
color color "#c0c0c0ff" Specifies the text color.
selcolor color "#ffffffff" Specifies the text color of the textbox in selected state.
direction "left" | "right" | "up" | "down" "left" Specifies the direction of the arrow.
check_selected boolean "false" If set to "true", navigation input events will only be send, if the arrow widget is in selected state.
Additionally all general and border attributes can be used.

Remarks

This element can be used as child within windows and widgets.
This element can only have one child widget.
