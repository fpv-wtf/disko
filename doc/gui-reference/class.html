Element class

Description

This element works only for <mmstheme>. With the special element <class> you can define various classes of the same base element. Each class has a unique name. This name can be used to connect an element within the dialog file to a special class defined in the theme file.

XML-Tag

<class>

Attributes

Attribute Description
name Unique name of the class.
type The type describes which base element the class represents. For example you can use type="mainwindow", type="label", type="menu" and so on (see elements). Based on the element type you can work with element-specific attributes.

Remarks

This element can only be used within <mmstheme>.

Examples

  • A <progressbar> theme class
