Mainwindow

Description

Specifies a mainwindow. A mainwindow resides within the static z-order of the disko window classes above the rootwindow and below the popupwindow Only one main window can be shown simultaneously.

XML-Tag

<mainwindow>

Attributes

All general window attributes can be used.

Remarks

This element can be used within <mmsdialog> and <mmstheme>.
This element can have a maximum of one child widget.

Related elements

<mmsdialog>, <mmstheme>, elements
