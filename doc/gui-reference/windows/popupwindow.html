Popupwindow

Description

Specifies a popupwindow. Popup windows will be shown above main windows and will be automatically hidden after the specified duration.

XML-Tag

<popupwindow>

Attributes

Attribute Domain Default Value Description
duration integer "5" Time in seconds the window is shown.If duration="0", it will not hide automatically.
Additionally all general window attributes can be used.

Remarks

This element can be used within <mmsdialog> and <mmstheme>.
This element can have a maximum of one child widget.

Related elements

<mmsdialog>, <mmstheme>, elements
