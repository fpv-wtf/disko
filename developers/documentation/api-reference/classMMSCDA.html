Detailed Description

Handles CDA playback.

Author:
Stefan Schwarzer (stefan.schwarzer@diskohq.org)

Matthias Hardt (matthias.hardt@diskohq.org)

This class is derived from MMSAV and specialized in handling CDA playback.
