Disko API Reference

Disko is a LGPL-licensed application framework, that can be used to develop GUI applications for embedded devices.

It uses different backends for graphics and it is specialised in touchscreen and remote control navigation. It provides a plugin based management component, which enables easy writing and extending embedded applications. Media playback (DVD, DVB, IPTV and audio streams) is provided by the well known xine media library or gstreamer.

For further information go to www.diskohq.com or directly to the documentation section.

A good starting point for the API reference is the overview of our modules.
