Downloads

If you don't want to compile the Disko sources yourself, you can download precompiled packages for Ubuntu and Debian. For installation instructions using the Ubuntu package manager and the Disko repository have a look at our installation guide.

To install Disko or the Disko demo application without using an package manager you can download the packages directly.
