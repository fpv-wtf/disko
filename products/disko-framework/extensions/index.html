Disko Framework Extensions

In the past developed Disko extensions and plugins are partly available for free or can be used under the terms of the Disko licence. These plugins are not an official part of Disko and their are only tested on some selected platforms.

Feature Description Platform Status License
Media Provider MMSMedia library for handling media data (Object Relational Mapping) using SQL. x86 Production LGPL
Soft Keyboard Screen keyboard with configurable layout. All Production LGPL

Plugins are runtime loadable Disko extensions.

Plugin Description Tested platforms Status Licence
Digital TV The functional package comprises 2 plugins. The TV Channels Import Plugin imports TV channels (IPTV, DVB-T, DVB-C, DVB-S). The Digital TV Player Plugin plays all digital TV programs (IPTV, DVB-T, DVB-C, DVB-S) based on the xine library. x86 (xine only), OMAP planned for 2010 (gstreamer) Production Disko
DVD The plugin plays a DVD based on the xine library. x86 (xine only), OMAP planned for 2010 (gstreamer) Production Disko
Photo Plugin for displaying pictures (e.g. from a digital camera). To begin with, the plugin shows all pictures as thumbnails and the user can decide whether they are to be displayed individually or as a slideshow. The user can also choose from different animations (picture transitions). x86 Outdated LGPL
MP3 Player The functional package comprises 2 plugins. The MP3 Import Plugin monitors one or several directories, loads the MP3 files stored in them and synchronizes the list with the Morphine.TV database. All available meta tags are also read and stored (TagLib). The MP3 Player Plugin is based on the xine library and plays back the data. x86 Outdated LGPL
EPG The functional package comprises 2 plugins. The EPG Import Plugin imports the EPG data supplied in the data stream of an active DVB-TV program into the media database (Morphine.TV). The EPG Plugin displays EPG data for expanding a TV plugin. All data from the media database is read out from the mmsmedia library and forwarded directly to the plugin. x86 Outdated LGPL
Video Player The functional package comprises 2 plugins. The Video Import Plugin monitors one or several directories, loads the video files stored in them and synchronizes the list with the Morphine.TV database. The Video Player Plugin plays back video data based on the xine library. All file formats can be played back that are also supported by the xine library (xine features). x86, OMAP (not tested), DaVinci (not tested) Outdated LGPL
Audio CD The plugin plays audio CDs based on the xine library. x86 Production Disko
pjsip SIP User Agent based on the PJSIP stack. The plugin comprises a dial pad and can be configured via an XML file. BF527, x86 Prototype GPL
minisip SIP User Agent based on the Minisip project. The plugin comprises a dial pad and can be configured via an XML file.
Please note, that not all SIP servers will be supported. Detailed informations are available on request. 		x86, BF527 Production GPL
RSS Import RSS files can be loaded from external systems and managed locally. x86 Outdated LGPL
